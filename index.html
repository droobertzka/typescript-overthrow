The Thing About TypeScript 🤔

  1. Limited ecosystem (definitely-typed)
    • ↳ #1 factor in language selection
  2. More decisions (+ mistakes)
  3. Limitations / struggles
    • ↳ polymorphism & generics (implied "any")
    • ↳ higher order functions / components
  4. Typing time & maintenance
  5. Steeper learning curve
    • ↳ limits on recruiting
    • ↳ training / onboarding
  6. Still JS at run time
